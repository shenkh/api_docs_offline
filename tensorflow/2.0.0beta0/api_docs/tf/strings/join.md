# tf.strings.join

Joins the strings in the given list of string tensors into one tensor;

### Aliases:

* `tf.compat.v1.string_join`
* `tf.compat.v1.strings.join`
* `tf.compat.v2.strings.join`
* `tf.strings.join`

``` python
tf.strings.join(
    inputs,
    separator='',
    name=None
)
```
