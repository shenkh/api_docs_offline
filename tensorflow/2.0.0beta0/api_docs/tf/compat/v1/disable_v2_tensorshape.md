# tf.compat.v1.disable_v2_tensorshape

Disables the V2 TensorShape behavior and reverts to V1 behavior.

``` python
tf.compat.v1.disable_v2_tensorshape()
```



Defined in [`python/framework/tensor_shape.py`](/code/stable/tensorflow/python/framework/tensor_shape.py).
